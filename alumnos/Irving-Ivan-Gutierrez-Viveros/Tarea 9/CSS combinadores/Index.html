Selectores en CSS

    Selectores básicos

  • * = Selecciona todos los elementos

    • Ejemplo: *{ margin: 0; }

  • elemento = Selecciona todos los elementos de ese tipo

    • Ejemplo: p { color: red; }

  • .clase = Selecciona elementos con esa clase

    • Ejemplo: .boton { ... }

  • #id = Selecciona un elemento por su id

    • Ejemplo: #menu { ... }

    Selectores combinadores

  • A B = Selecciona B dentro de A (descendiente)

    • Ejemplo: div p {}

  • A > B = Selecciona B hijo directo de A

    • Ejemplo: ul > li {}

  • A + B = Selecciona B que está justo después de A (hermano siguiente)

    • Ejemplo: h1 + p {}

  • A ~ B = Selecciona todos los B hermanos posteriores de A(hermano siguiente)

    • Ejemplo: h1 ~ p {}

    Selectores de Pseudoclases

  • :hover = Actua cuando el mouse pasa por encima

    • Ejemplo: a:hover {}

  • :first-child = Actua sobre el primer hijo

    • Ejemplo: li:first-child {}

  • :last-child = Actua sobre el ultimo hijo

    • Ejemplo: li:last-child {}

  • :nth-child(n) = Elemento en la posición n

    • Ejemplo: tr:nth-child(odd) {}

  • :checked = Genera un elemento marcado (checkbox, radio)

    • Ejemplo: input:checked {}

    Selectores de Pseudoelementos

  • ::before = Inserta contenido antes

    • Ejemplo: p::before { content: "*"; }

  • ::after = Inserta contenido despues

    • Ejemplo: p::after { content: "."; }

  • ::first-letter = Selecciona la primera letra

    • Ejemplo: p::first-letter {}

  • ::first-line = Selecciona la primera línea

    • Ejemplo: p::first-line {}

    Selectores de atributo

  • [attr] = Selecciona todos los elementos que tienen ese atributo (sin importar su valor)

    • Ejemplo: input[placeholder] {border: 2px solid blue;}

  • [attr="valor"] = Selecciona elementos cuyo atributo tenga ese valor exacto

    • Ejemplo: input[type="password"] {border: 2px solid red;}

  • [attr^="valor"] = Selecciona elementos donde el valor del atributo empieza por cierto texto

    • Ejemplo: a[href^="https"] {color: green;}

  • [attr$="valor"] = Selecciona elementos donde el valor del atributo termina en cierto texto

    • Ejemplo: img[src$=".jpg"] {border: 3px solid orange;}

  • [attr*="valor"] = Selecciona elementos donde el valor del atributo contiene cierto texto (en cualquier parte)

    • Ejemplo: a[href*="contacto"] {font-weight: bold;}
